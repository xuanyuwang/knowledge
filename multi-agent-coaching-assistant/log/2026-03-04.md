### Criterion Display Name Resolution

**Problem:** `RetrieveCoachingProgresses` returns empty `criterion_display_name` for focus criteria, so the app fell back to `criterion_id` (a UUID like `3800f0e9-f3cf-4013-acf7-ff093ae7edd7`).

**Solution:** Added `_fetch_criterion_display_names()` helper in `app.py`:
1. Collects unique `scorecard_template_name` values from coaching progress data
2. Calls `ListCurrentScorecardTemplates` (with `VIEW_BASIC_WITHOUT_STATS`) to fetch templates for the profile
3. Parses each template's `template` Struct (a JSON blob) to extract `{criterion_id: displayName}` mapping
4. Handles both V1 (`criteria` array) and V2 (`items` array with nested chapters) template formats
5. Recursively traverses chapters and branches for nested criteria
6. Applied before scoring, so display names propagate through to assessments and analysis text

**Template JSON structure:**
- V1: `{"criteria": [{"identifier": "uuid", "displayName": "Name", ...}]}`
- V2: `{"version": 2, "items": [{"identifier": "chapter-id", "displayName": "Chapter", "items": [{"identifier": "uuid", "displayName": "Criterion Name", ...}]}]}`

**Result:** Criterion names now show e.g. "Accurate", "Kind", "Straighforward" instead of UUIDs.

### `/api/recommend-direct` Endpoint

**Motivation:** `/api/recommend` depends entirely on `/api/identify` output. For programmatic callers, two sequential HTTP calls is unnecessary friction.

**Request** (same as `/api/identify`):
```json
{"agent_names": ["customers/cresta/users/..."], "usecase_name": ""}
```

**Response** (merged identify + recommend):
```json
{
  "summary": "LLM team coaching summary",
  "recommendations": [...],
  "agents_analyzed": 5,
  "assessments": [...],
  "identify_summary": "5 agents assessed, 2 critical, 1 high, 2 low"
}
```

**Tested on staging:**
```bash
curl -s -X POST \
  -H "Authorization: Bearer ${TOKEN}" \
  -H "X-Cluster-Id: voice-staging" -H "X-Customer-Id: cresta" -H "X-Profile-Id: walter-dev" \
  -H "Content-Type: application/json" \
  -d '{"agent_names": ["customers/cresta/users/231dd765a58558ae"]}' \
  https://coaching-ai-summary.voice-staging.internal.cresta.ai/api/recommend-direct
```
Returns assessments + LLM recommendations in a single call. Verified working on revision 15.
