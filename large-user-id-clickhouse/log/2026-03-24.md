## Progress

- This causes the usecase filter to receive nil -> `usecase_id IN (NULL)` -> 0 rows
- Regression caused by CONVI-6316 (always pass FinalUsers via ext table) + ENABLE_EXT_TABLE_FOR_USER_FILTER enabled globally
