## Details

### Agent Assist backward compatibility

The Agent Assist page has 24 `useInsightsRequestParams` call sites across 11 files, plus 5 in `useGetHintStatsByHintType`. None pass `filterToAgentsOnly`. Rather than adding the option to every file, defaulted to `true` at the hook level:

```ts
filterToAgentsOnly: filterOptions?.filterToAgentsOnly ?? true,
```

This preserves current behavior (BE hardcodes `true`) while callers that explicitly pass `false` (Leaderboard Manager tab) are unaffected.
