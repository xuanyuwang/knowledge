### FE Code Investigation

The template builder is a 2-tab form. For NAScore we only touch tab 1 (Configure Criteria).

Key component: `CriteriaLabeledOptions.tsx` renders the options table for labeled-radios and dropdown criterion types. It has:
- Options array (label/value pairs) via `useFieldArray`
- Scores array (separate score values) via `useFieldArray` — gated by `enableDuplicateScoreForCriteria` flag
- "Allow N/A" checkbox bound to `settings.showNA` via `useController`

The N/A checkbox currently only toggles the boolean — no UI for configuring N/A score value.
