# Knowledge (Senior → Staff Track)

This repo exists to make my growth toward **Staff Engineer** explicit, measurable, and repeatable.

## What I’m using this repo for

- Track where I’m operating at *senior* vs where I need to operate at *staff*
- Capture “staff-level artifacts”: problem framing, options, tradeoffs, rollout plans, decision records, and postmortems
- Maintain a small set of high-leverage projects that create cross-team impact (not just local feature delivery)
